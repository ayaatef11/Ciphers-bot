Cipher Bot is a browser-based JavaScript application that demonstrates the implementation of classical and modern cryptographic algorithms for educational purposes.  
All encryption and decryption algorithms in this project are implemented **from scratch** without relying on external cryptographic libraries.

This repository is designed to help students and beginners understand how encryption algorithms work internally by providing clear, readable implementations.

## Features

- Pure JavaScript implementations (no crypto libraries)
- Interactive chat-style user interface
- Cipher-specific key validation
- Supports both encryption and decryption modes
- Easy switching between different ciphers
- Designed for educational and academic use

---

##  Usage

1. Select a cipher from the available list.
2. Enter your plaintext or ciphertext.
3. Provide the required encryption key.
4. Click **Encrypt** or **Decrypt**.
5. View the result instantly in the chat interface.
